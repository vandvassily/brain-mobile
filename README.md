# 组件库

移动端业务组件库

## 背景

虽然移动端产品已经统一使用 UI 基础库 `vant` ，不同的产品下有各自的业务组件，但是不同的产品业务中会遇到重复的功能。粗暴的复制业务组件代码，不利于之后维护。现在的想法是参考 `vant` 的业务组件写法，构建公司自己的业务组件库，方便进行版本管理、代码维护、以及单元测试

技术：`vant-cli` 使用有赞开源的工具，拒绝造轮子。

### vant-cli 工具链说明

vant-cli 介绍

1. 说明文档
2. 单测
3. 按需引入（es module）
